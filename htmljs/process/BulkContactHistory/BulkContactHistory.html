To select multiple items in a list hold the CTRL key and click the options you want. To un-select options hold CTRL and click that option.
There is no undo for this tool. Please use caution when using this tool. To reverse a mistake you will need to go in to each contact and edit/delete/change the record in question.
Please be cautious when using the PD Attendees option. It only makes sense to use this tool for attendees when all schools in the list have the same number of attendees.
